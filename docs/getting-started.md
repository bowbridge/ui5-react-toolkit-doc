# Getting Started

### Installation

_npm_

```shell
npm i @bowbridge/ui5-react-toolkit
```

_yarn_

```shell
yarn add @bowbridge/ui5-react-toolkit
```

_Peer Dependencies_

```shell
yarn add yup react-hook-form @hookform/resolvers

or

npm i yup react-hook-form @hookform/resolvers
```
